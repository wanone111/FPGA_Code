======================
L3 Overlay User Guide
======================

The code within the L3 overlay attempts to provide an easy-to-use abstraction of any OpenCL™ calls and pre/post-processing that is required to utilize FPGA acceleration kernels.

Currently, one solution is provided: regular expression match acceleration.
